Política de Privacidade — Next Episode

Última atualização: 9 de julho de 2026

Este documento explica quais dados o aplicativo Next Episode coleta, para que servem e como você pode solicitar a exclusão da sua conta e dos seus dados.

Quem somos

O Next Episode é um aplicativo desenvolvido de forma independente para acompanhamento de séries de TV. Contato do responsável pelo app: nagibneto_2@hotmail.com.

Quais dados coletamos

  • Dados de cadastro: e-mail, nome de usuário e apelido. A senha é gerenciada pelo Supabase Auth e nunca é armazenada em texto puro nem acessível por nós.
  • Atividade no app: séries que você segue, episódios marcados como assistidos, notas dadas a episódios.
  • Conteúdo que você publica: comentários em episódios e imagens/GIFs anexados a eles.
  • Relações sociais: pedidos de amizade enviados/aceitos e quem você segue no app.
  • Notificações: um token de dispositivo (via Expo/Firebase) para enviar avisos de novos episódios, caso você autorize notificações.

Não coletamos dados de localização, contatos, nem usamos rastreamento publicitário de terceiros. Não vendemos dados pessoais.

Com quem compartilhamos

  • Supabase — hospeda o banco de dados, autenticação e armazenamento de imagens do app.
  • TMDB (The Movie Database) — fornece informações públicas sobre séries e episódios (sinopses, datas, pôsteres). Suas buscas por séries podem ser enviadas à TMDB, sem dados pessoais identificáveis.
  • Expo / Firebase Cloud Messaging — usados apenas para entregar notificações push, se você autorizar.

Nenhum desses serviços recebe seus dados para fins de publicidade.

Por quanto tempo guardamos seus dados

Enquanto sua conta existir. Você pode excluir sua conta e todos os dados associados a qualquer momento — veja abaixo como.

Como excluir sua conta e seus dados

Pelo app: abra o Next Episode, vá em Perfil → Excluir conta e confirme. Isso apaga permanentemente seu perfil, séries seguidas, episódios assistidos, notas, comentários, imagens enviadas, amizades e token de notificação.

Sem acesso ao app: envie um e-mail para nagibneto_2@hotmail.com a partir do endereço cadastrado na conta, pedindo a exclusão. Vamos confirmar e remover seus dados em até 30 dias.

Segurança

O banco de dados usa Row Level Security (RLS): cada usuário só acessa e altera os próprios dados privados, e as regras de acesso a dados compartilhados (como o feed de amigos) são reforçadas no banco, não só no aplicativo.

Crianças

O Next Episode não é direcionado a crianças menores de 13 anos e não coletamos intencionalmente dados desse público.

Alterações nesta política

Podemos atualizar esta página quando o app mudar. A data no topo indica a versão mais recente.
